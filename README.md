Este é um projeto desenvolvido em Python que implementa um sistema simples de cadastro, login e recuperação de conta de usuários. O sistema utiliza estruturas básicas como listas para armazenar os dados e validações rigorosas para garantir a integridade das informações inseridas.

**Funcionalidades**

 1. Cadastro de Usuário: Permite registrar nome, email, senha e CPF com validações específicas.
 
 2. Login de Usuário: Autentica o usuário com base no email e senha cadastrados.
 
 3. Recuperação de Conta: Simula o envio de código de recuperação para emails válidos.
 
 4. Sair do Sistema: Encerra a execução do programa de forma segura.

**Como Executar no VS Code**
1. Certifique de ter e a extensão Python for VS Code instalados.

2. Abra a pasta do projeto no Visual Studio Code.

3. Abra o arquivo cadastro_de_usuario.py.

4. Clique no botão "Run"  no canto superior direito ou pressione F5 para executar.

5. Siga as instruções exibidas no terminal integrado do VS Code.
</div>

 **Validações Implementadas**
 
1. Email: Deve conter domínio válido (@gmail.com, @hotmail.com ou @outlook.com) e não pode estar duplicado.

2. Senha: Deve ter no mínimo 6 caracteres.

3. CPF: Deve conter exatamente 11 dígitos numéricos e não pode estar duplicado.

4. Nome: Não pode estar vazio.

**Observações**
Os dados são armazenados temporariamente em memória (listas), portanto, serão perdidos ao encerrar o programa.
Este projeto tem fins educacionais e demonstra domínio de lógica de programação, estruturas de repetição (while) e condicionais (if/elif/else).
